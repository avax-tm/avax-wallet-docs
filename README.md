## Why AVAX Wallet

Avalanche runs three specialized blockchains: the C-Chain for smart contracts and DeFi, the X-Chain for fast asset transfers, and the P-Chain for staking and validation. Most wallets — MetaMask, Trust Wallet, Exodus, Coinbase Wallet — only support the C-Chain. That means one chain out of three.

The [official Avalanche Wallet was deprecated](https://avaxwallet.net/guides/what-happened-to-avalanche-wallet) in March 2024. Its replacement, Core, is a multichain product that serves Ethereum, Bitcoin, and other EVM chains alongside Avalanche. Users who wanted a focused, dedicated AVAX experience lost their tool.

AVAX Wallet v0.1.5 picks up where the original left off. Built from the same open-source codebase (BSD-3-Clause), with a rebuilt transaction layer for Durango network compatibility. Non-custodial. Desktop-first. Avalanche-only. The wallet supports [native AVAX staking](https://avaxwallet.net/staking), a [built-in Ethereum bridge](https://avaxwallet.net/bridge), and [guided cross-chain transfers](https://avaxwallet.net/avalanche-chains) that prevent wrong-network errors.

## About

AVAX Wallet is an open-source, non-custodial desktop wallet for the Avalanche network. It is built from the original Avalanche Wallet codebase (ava-labs/avalanche-wallet) under the BSD-3-Clause license, with a rebuilt transaction layer for Durango network compatibility.

Version 0.1.5 supports all three Avalanche chains (X, C, P), native staking, cross-chain transfers, and an Ethereum bridge. Available for Windows, macOS, and Linux.
